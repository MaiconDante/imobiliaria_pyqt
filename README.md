# Imobiliária PyQt

Projeto desenvolvido para estudos da Pós Graduação Latu Sensu em Desenvolvimento de Sistemas em Python, realizada na UNICESUMAR de PyQt6.

## Objetivo

Criar um sistema simples de cadastro de imóveis para aprendizado de:

- PyQt6
- SQLite
- Git e GitHub
- Estruturação de projetos
- Boas práticas de desenvolvimento

## Funcionalidades

- Cadastro de imóveis
- Listagem de imóveis
- Edição de imóveis
- Exclusão de imóveis

## Tecnologias

- Python
- PyQt6
- SQLite
